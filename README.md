# 🏊 Piscine Projects

This repository contains exercises and projects completed during the **Piscine program**.

The Piscine is an intensive coding bootcamp where students practice programming fundamentals through daily exercises and projects.

## 📚 Topics Covered

* Shell scripting
* C programming
* Basic algorithms
* Problem solving
* Unix commands

```

```

Each folder contains exercises related to that specific module.

## 🎯 Goal

The goal of these exercises is to practice:

* programming logic
* working in a terminal environment
* writing clean and efficient C code
* understanding low-level programming concepts
